Question 1: What is JavaScript?
JavaScript is a programming language that allows you to add interactivity and dynamic features to websites.
Question 2: How do you declare a variable in JavaScript?
You can declare a variable in JavaScript using the 'var', 'let', or 'const' keywords.
Question 3: What is an event handler?
An event handler is a function that gets executed in response to a specific event, such as a button click or a key press.
